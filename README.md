# Estudos de Python para Análise de Dados 📊🐍  

Este repositório contém materiais, códigos e projetos desenvolvidos durante meus estudos de **Python**, com foco em **análise de dados** utilizando as bibliotecas **NumPy** e **Pandas**. O objetivo é explorar diferentes técnicas de manipulação de dados, estatísticas e boas práticas para trabalhar com grandes volumes de informações.  

## 📌 Objetivos  

- Aprender e consolidar conceitos fundamentais do **Python**.  
- Explorar as funcionalidades das bibliotecas **NumPy** e **Pandas**.  
- Realizar **manipulação de dados**, limpeza e transformação de datasets.  
- Aplicar **estatísticas básicas** e métodos exploratórios para análise de dados.  
- Criar notebooks interativos e práticos para estudo e experimentação.  

## 🛠 Tecnologias Utilizadas  

- **Python** 3.13 
- **NumPy** – Para cálculos numéricos e operações vetoriais/matriciais.  
- **Pandas** – Para manipulação e análise de dados.  
- **Jupyter Notebook** – Para organização dos estudos e execução interativa dos códigos.
